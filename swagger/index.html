User-managed items that can be put into bins.
Items are classes of concrete BinItems that reside in bins. They are per-user; there is no common set of Items available. An Item must be created before it can be put into a bin and an arbitrary number of copies of the item can be put into an arbitrary number of bins.

Items are defined by a name and have a mandatory measurement, ie. "Cereal 300g"
